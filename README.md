# NBUT Online Judge V3

Ningbo University of Technology Online Judge System V3 Web.

## 开发阶段准备

依赖：

+ [Node.js](https://nodejs.org/en/) > 4
+ [Bower](http://bower.io/)
+ [MySQL](https://www.mysql.com/)
+ [Memcached](http://memcached.org/)

### Node.js 依赖

```sh
$ make install
```

### 构建前端文件

```sh
$ make dev
```

开发时候请始终开启前端构建监视：

```sh
$ make watch
```
